Examples

(Images are clickable examples)

To view the entire JSON collection:

http://eliteagility.diskstation.me:8080/movies/

A collection resource in JSON can be delivered in the following format:

http://eliteagility.diskstation.me:8080/movies/category/[CATEGORY]/year/[YEAR]

The API is not restricted to the above. The items can be used in any order, such like the following:

http://eliteagility.diskstation.me:8080/movies/winner/category/[CATEGORY]

http://eliteagility.diskstation.me:8080/movies/year/[YEAR]

A singleton resource in JSON can be delivered in the following format:

http://eliteagility.diskstation.me:8080/movies/category/[CATEGORY]/year/[YEAR]/winner

Like with the collection resource. the API is not restricted to the above.The items can be used in any order, such like the following:

http://eliteagility.diskstation.me:8080/movies/winner/year/[YEAR]/category/[CATEGORY]

A parameterized search can also be done in the following formats:

http://eliteagility.diskstation.me:8080/search?category=[CATEGORY]&year=[YEAR]

http://eliteagility.diskstation.me:8080/search?year=[YEAR]
